# Design Analysis and Algorithm Repository

A collection of Design and Analysis of Algorithms (DAA) implementations, including sorting, searching, graph algorithms, dynamic programming, greedy techniques, and complexity analysis in C++ and Jupyter Notebook.

## 📚 Contents

### Searching Algorithms
- **Linear Search** (`Linear.ipynb`) - Sequential search with O(n) time complexity
- **Interpolation Search** (`Interploation_Search.ipynb`) - Improved search for uniformly distributed sorted arrays
- **Interpolation Search Practice** (`Practice_Interploation_Search_Problem_Statement_3.ipynb`) - Hands-on problem solving

### Coming Soon
- Sorting Algorithms (Quick Sort, Merge Sort, Heap Sort)
- Graph Algorithms (BFS, DFS, Dijkstra, Floyd-Warshall)
- Dynamic Programming Problems
- Greedy Algorithms
- Complexity Analysis

## 🚀 Getting Started

### Prerequisites
- Python 3.x with Jupyter Notebook
- C++ compiler (for future C++ implementations)

### Running the Code
1. Clone the repository:
```bash
git clone https://github.com/Surya1234-git/Design-Analysis-and-Algorithm-.git
```

2. Launch Jupyter Notebook:
```bash
jupyter notebook
```

3. Open any `.ipynb` file to run the algorithms.
